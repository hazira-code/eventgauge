# EventGauge 📊

## Smart Event Feedback & Analytics Platform

EventGauge is a modern web application designed to collect, analyze, and visualize event feedback in real time. The platform helps organizers understand attendee satisfaction, identify improvement areas, and make data-driven decisions for future events.

---

## 🚀 Features

### Feedback Collection

* Easy-to-use feedback forms
* Event-specific surveys
* Rating-based and descriptive feedback
* Mobile-friendly interface

### Real-Time Analytics

* Live feedback tracking
* Average rating calculation
* Satisfaction metrics
* Interactive dashboards

### Data Visualization

* Pie charts
* Bar graphs
* Trend analysis
* Feedback distribution reports

### User Management

* Secure authentication
* Admin dashboard
* Event organizer access
* Participant feedback submission

### Reporting

* Event performance reports
* Downloadable analytics
* Feedback summaries
* Improvement recommendations

---

## 🎯 Problem Statement

Event organizers often struggle to collect and analyze attendee feedback efficiently. Traditional feedback methods are slow, difficult to manage, and provide limited insights.

EventGauge solves this problem by providing a centralized platform for collecting feedback and generating actionable analytics.

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
cd EventGauge
```

### Install Dependencies

```bash
npm install
```

### Start Backend

```bash
npm run server
```

### Start Frontend

```bash
npm start
```

## 📋 Workflow

1. Organizer creates an event.
2. Participants submit feedback.
3. Feedback is stored securely.
4. Analytics engine processes responses.
5. Dashboard displays insights.
6. Organizers review reports and recommendations.
